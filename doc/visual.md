# Visual Bible

## 1. Visual thesis

《菌落封锁》是一场清新、可读的培养皿净化实验：薄荷培养液中的净化球安静发光，随机生成的菌株以强轮廓和明确运动制造短促压力。核心承诺是“先看懂 DNA 行为，再迅速落球”。

## 2. Composition and camera

- 390×720 固定竖屏场地；培养皿直径 430 px，可从左右出画。
- HUD 固定在顶部 24–150 px；DNA 编号置于皿上方，底部保留水印与声音控制。
- 培养皿内为中央 4×6 净化球阵列与左右各 4 颗外围球；所有相邻球心横向间距均为 62 px。

## 3. Color

- 背景：深海蓝 #0D2235 / #17394D；培养液：薄荷青；净化球：暖白至冰青渐层。
- DNA 色素：珊瑚、雾紫、琥珀、荧光绿。它们只用于菌株与 DNA 条，不承担唯一危险提示。
- 菌株始终带深色边缘、实体填充与投影，和半透明净化球保持分层。

## 4. Typography

- 统一使用 Avenir Next、PingFang SC、Microsoft YaHei、system-ui。
- 标题 34 px/800；HUD 数字 28 px/800；DNA 编号 9 px/800、字距 0.12em。
- 培养结算优先显示当前菌株编码、名称与行为记录。

## 5. Shape, material, and lighting

- 培养皿为正圆暖白玻璃，内层薄荷培养液有高光、内阴影与虚线刻度。
- 净化球为 52 px 实体胶质圆体，按下下沉 3 px。
- 菌株形体为实心杆、链、弧、孢；弧菌不允许透明，必须以实心渐层区分净化球。

## 6. Characters, environments, and assets

- 背景实验室物件仅有低对比移液器与样本管，不能遮挡培养皿。
- 无餐具、食物、筷子或拟真医疗器械；菌株不拟人化、不带脸。

## 7. UI and icons

- 使用 Lucide 线性图标，仅用于声音控制；禁止 Emoji 充当功能图标。
- 主按钮为 56 px 高的薄荷胶囊；覆盖层为圆角 28 px 的半透明实验记录。
- 触控目标均不小于 44×44 px；游戏动作用 PointerDown。

## 8. Motion and VFX

- 净化球命中：同帧涟漪与 6 个微粒，460 ms 结束。
- DNA 动态：滑行、呼吸、抖动或闪烁；路线为横向、纵向、环形或跳格。
- 分裂菌每 1.8 秒有一段 650 ms 的虚线环预警，预警后扩展至最多 6 格。
- reduced-motion 下将循环动画收敛为静态形态与清晰的状态色。

## 9. Translated references

采用《急刹车》式的清爽深蓝、冰青与圆润控制感，但不复刻其冰原场景或组件。实验室感来自玻璃、培养液、编号与克制仪器轮廓，而非医疗写实。

## 10. Anti-patterns

- 禁止透明危险物、矩形净化球、厚黑描边与任意餐饮符号。
- 禁止仅靠颜色提示危险；每个菌株还必须有实体轮廓、运动类型与路线差异。
- 禁止把随机 DNA 伪装成固定五关顺序。

## 11. Vertical-slice acceptance

- 开始：2 秒内可读出“下净化球，封住菌株”。
- 进行中：显示 DNA 周期编号、随机菌株编码和“封锁进度 0/12”；危险菌和净化球一眼可分。
- 成功：第 12 颗净化球落下即显示刚完成的菌株名、DNA 编码、行为记录与分数。
- 超时：显示“封锁不足”和实际完成进度，不能把未操作或未完成净化误判为成功。
- 失败：显示“碰到菌株！”，并保留当前菌株以便重试。
- 390×720：培养皿、32 球、HUD、按钮不裁切，外围球与中间列距一致。

## 12. DNA system

每轮菌株从 5 组独立 DNA 组合而成：

| DNA | 可选值 | 可见与玩法结果 |
| --- | --- | --- |
| 色素 | 珊瑚 / 雾紫 / 琥珀 / 荧光绿 | 实体填充、高光、DNA 条色 |
| 形体 | 杆 / 链 / 弧 / 孢 | 危险占格 4 / 4 / 3 / 1 |
| 动态 | 滑行 / 呼吸 / 抖动 / 闪烁 | 运动节奏 |
| 路线 | 横向 / 纵向 / 环形 / 跳格 | 移动方向与转折 |
| 分裂 | 无 / 二分裂 | 是否产生带预警的额外危险格 |

培养成功后随机生成下一株，且至少改变一个外显 DNA；污染失败时保持当前组合。
