# 《别踩线！》设计方向评审

此页仅用于选择视觉方向，不改变现有玩法或正式视觉规范。参考《急刹车》时，只借用其清新而克制的色彩组织、场景优先的层级和低干扰 HUD，不复刻冰原、角色、构图或组件。

## 01 海风汤面（暂选）

- 深蓝夜色、薄荷青、珊瑚红和暖白。
- 最接近《急刹车》的清爽结构：碗像浅水中的主体，HUD 轻而明确。
- 风险：夜色若处理不好会显得偏冷，需要用暖白汤面保持食物温度。

## 02 柚子食堂

- 奶油白、薄荷绿、柚子橘与深青灰。
- 最轻快、最适合连续短局的一版；红面危险色的辨识度最高。
- 风险：需要克制装饰，避免变成泛餐饮品牌页面。

## 03 青瓷小馆

- 青瓷绿、米纸白、朱砂红和深松石。
- 最安静、最有细节质感；信息密度最低。
- 风险：调性较克制，连击奖励需要额外设计为更明显的高光时刻。
