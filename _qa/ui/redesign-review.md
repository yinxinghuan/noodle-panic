# UI Redesign Review — 2026-07-21

## Scope

将旧版“深蓝电视综艺餐盘”改为深夜拉面吧台。保留 4×6 点击玩法、计分、连击和结算，替换场景、HUD、开始页、结算页、盘面材质与全部动效语言。

## Evidence reviewed

| 状态 | 视口 | 结果 |
|---|---:|---|
| 开始菜单 | 390×720 | 米纸菜单、标题、倒计时、汤碗和主按钮均完整；主动作清楚。 |
| 游戏中 | 390×720 | 汤碗边界、红色危险面与安全汤格有明确区分；HUD 位于碗外。 |
| 时间结算 | 390×720 | 收工信息、分数与重开动作位于同一张米纸菜单。 |
| 窄屏 | 320×568 | 场景整体缩放，标题、碗、提示与静音按钮均未裁切。 |

## Findings and resolution

- P1（旧版）：厚描边统计卡、随机彩带和大遮罩竞争注意力，餐盘没有真实材质。已移除，改为低对比文字 HUD、黑漆桌面、粗陶碗与暖光汤面。
- P1（旧版）：开始与结算页像与玩法无关的通用叠层。已改为统一的米纸菜单，复用同一边角、阴影、纸张色温和按钮材质。
- P2（旧版）：普通点击/连击的效果语言不一致。已统一为汤面涟漪、金色安全印、暖金碗沿和蒸汽增强。
